Clustering

Description

Clustering is the method of segregating the data into a certain number of groups compromising of similar or equivalent data points. Points within the similar cluster are as same as possible whereas points belonging to another cluster are as completely different or dissimilar as possible. An example where clustering is used in our daily lives would be in shopping malls where items of the same category are places under a particular section and they do not overlap so if someone wants a particular product they can easily navigate to the particular section of their own requirement. Clusters can also have inner clusters or say smaller groups as items in the dairy section has milk section, butter section, cheese section. Clustering significantly helps in management of data in a better way. So here in our model we are analysing the accident behaviours upon different scenarios and given condition to get a detailed data analysis.

K-means Clustering

Description

K-Means which can be defined as a non-supervised machine learning algorithm which is used in clustering the given dataset into k sets of clusters. It’s a non-parametric clustering method. K means algorithm is used in scenario where we want to make groups of similar things from randomly distributed items or data points. One of the most common method used to find the clusters and label in a data set using a predefined value of K. So, to get an idea of the actual number of clusters present in the data we need to calculate iteratively for a number of clusters and one by one compare the result.
